# Civic Infrastructure: Am I Ready for an EV?

This project is an interactive data-visualisation website about the UK's public EV charging network. It explores how charging infrastructure has grown over time, whether local supply is adequate relative to likely demand, and how residents might experience charging accessibility at neighbourhood scale.

## Project Structure

The site is organised into five main pages:

- `Overview`: introduces the overall story and the structure of the dashboard.
- `Part 1 - Growth`: shows national charging-network growth, policy milestones, regional comparison, and city-level charging distribution.
- `Part 2 - Adequacy`: compares charger supply and likely local dependence on public charging across London boroughs and selected city areas.
- `Part 3 - Tool`: provides an interactive resident-facing exploration tool for testing local charging accessibility under different assumptions.
- `About`: summarises contributors, methods, data sources, and the technical stack.

## Main Features

- Editorial-style landing page and multi-page navigation
- National EV charging growth charts and timeline storytelling
- Regional and city-level interactive maps
- Borough-level adequacy comparison and ranking views
- Neighbourhood charging accessibility tool with map interaction, filtering, and search

## Data and Methods

The project combines several public datasets and map sources, including:

- UK Department for Transport charging statistics
- UK vehicle registration and EV-related indicators
- OpenStreetMap charging-location data
- Open Charge Map station metadata
- ONS boundary and census-related geographic data

These datasets are used to build a three-part analytical story, moving from national growth to local adequacy and finally to neighbourhood-level accessibility.
